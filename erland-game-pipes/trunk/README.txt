2. LIBRARIES
============

3. PREREQUISITES
================
- Java 1.4 or later must be installed

4. FILES
========
This archive should contain the following files:
- readme.txt (this file)
- *.jar (The application code)
- Pipes.jnlp (The webstart startup file)
- Pipes.bat (A bat file to start the application)

5. CONFIGURING JNLP
===================
The jnlp-file contains references to my local machie, so you will have to
change these to your machine

6. RUN
=================
To start the application just lanunch the included
jnlp file with webstart or the included bat-file to start the application.
